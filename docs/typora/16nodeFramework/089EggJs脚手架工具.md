# 089 EggJs脚手架工具

安装脚手架：在npmjs中搜索`egg-init`包。

```bash
cnpm install egg-init -g
```

创建项目

```bash
npm init egg --type-simple 
# simple也可为typescript意思是使用ts语言
```

安装相关依赖

```bash
npm install
```



