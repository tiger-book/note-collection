# 01 常用单位px

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>01-前端开发常用单位-像素</title>
    <style>
        *{
            margin: 0;
            padding: 0;
        }
        div{
            width: 100px;
            height: 100px;
            background: red;
        }
    </style>
</head>
<body>
<div></div>
<script>
    /*
    1.什么是像素(Pixel)?
    在前端开发中视口的水平方向和垂直方向是由很多小方格组成的, 一个小方格就是一个像素
    例如div尺寸是100 x 100, 那么水平方向就占用100个小方格, 垂直方向就占用100个小方格

    2.像素特点
    不会随着视口大小的变化而变化, 像素是一个固定的单位(绝对单位)
    * */
</script>
</body>
</html>
```
