# 04 变量插值

## 01 叙述

```
1.什么是变量插值?
如果是属性的取值可以直接使用变量,
但是如果是属性名称或者选择器名称并不能直接使用变量, 必须使用变量插值的格式

2.SASS中的变量插值
SASS中的变量插值和LESS中也一样, 只不过格式不一样
LESS变量插值格式: @{变量名称}
SASS变量插值格式: #{$变量名称}
```



## 02 示例

```scss
$size: 200px;
//属性名称
$w: width;
//选择器
$s: div;

#{$s}{
  #{$w}: $size;
  height: $size;
  background: red;
}
```
